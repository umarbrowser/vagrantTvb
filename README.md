# Vagrantfile Scrpt
> Simple vagrantfile script and bash script for virtual brain under INCF, for GSoC 2018 prove.

This will install TVB package contain debian jessie on virtualbox.

![](header.png)

## Installation

OS X & Linux:

```sh
vagrant up
```

Windows:

```sh
vagrant up
```
